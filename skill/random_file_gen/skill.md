# Random File Generator

**Deskripsi**: Skill untuk membuat file teks dengan konten random untuk keperluan testing atau placeholder.
**Author**: EMORA Skill Factory
**Versi**: 1.0.0

## Trigger
Gunakan skill ini ketika user meminta pembuatan file dummy, file testing, atau konten random secara cepat.

## Langkah-langkah
1. Tentukan nama file dan jumlah baris yang diinginkan.
2. Generate string random atau teks placeholder.
3. Gunakan tool `write_file` untuk menyimpan konten tersebut ke dalam file.

## Contoh Penggunaan
"Bro, buatin file dummy random namanya test.txt isinya 10 baris teks asal."

## Catatan
Skill ini hanya menghasilkan teks random sederhana, bukan data terstruktur seperti JSON atau CSV kecuali diminta spesifik.
